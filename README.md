## Course Evaluation Script
这个脚本可以自动填充课程网站评价系统的表单。
### Guide
首先下载这个repo,
```shell
$ git clone https://github.com/Ref-rain/course-evaluation.git
```
#### 打开谷歌浏览器的扩展程序
![step1](./img/step1.png)
#### 打开开发者模式
![step2](./img/step2.jpg)
#### 点击"加载已解压的的扩展程序",将下载的文件夹加载进来
![step3](./img/step3.jpg)
#### 加载成功后
